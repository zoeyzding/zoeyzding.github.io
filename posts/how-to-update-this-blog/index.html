How to update this blog (built with Hugo & GitHub Pages)

July 29, 2021 · zoeyzding

#1 Write a post#

Open cmd，cd to hugo blog local folder。

hugo new posts/"new post.md"

Open “new post”. Start wrting by using a markdown editor (recommended).

#2 Preview#

hugo server -D

Make sure everything works fine. Change draft:true to draft:false in the metadata section of the post.

#3 Git update#

  1. Run hugo
hugo

  1. Go to “public” folder, push to GitHub.

    cd public
git add .
git commit -m "xxxxx"
(git pull)
git push

  2. Go to the blog url to see the final result.
